## 项目介绍

《東耳篮球馆会员信息管理系统》是一个基于Spring Boot框架开发的篮球馆会员管理平台。本项目旨在为篮球馆提供便捷高效的会员信息管理服务，实现会员信息、场地预约、消费记录、会员积分等功能的一体化管理。

## 项目功能

1. **会员管理**：支持添加、修改、查询和删除会员信息，包括会员姓名、联系方式、会员卡号等。
2. **场地预约**：会员可以在线预约篮球场地，系统实时更新场地使用情况，避免冲突。
3. **消费记录**：记录会员的消费信息，如场地租赁、会员卡充值等，方便财务管理和会员查询。
4. **会员积分**：根据会员的消费金额和次数，自动计算积分，积分可兑换礼品或享受优惠。
5. **权限控制**：实现管理员和普通会员的权限区分，保障系统安全。

## 技术栈

1. **前端**：
   - HTML/CSS：构建页面布局和样式。
   - JavaScript：实现页面交互逻辑。
   - Vue.js：前端框架，提高开发效率。

2. **后端**：
   - Spring Boot：基于Spring框架的微服务开发框架，简化项目配置和部署。
   - MyBatis：数据持久层框架，实现数据库操作。
   - MySQL：关系型数据库，存储会员、场地等数据。
   - Spring Security：实现用户权限管理和安全认证。

3. **开发工具和环境**：
   - IntelliJ IDEA：集成开发环境。
   - Maven：项目构建和管理工具。
   - Git：版本控制工具。
   - CentOS：服务器操作系统。

本项目采用前后端分离的开发模式，前端负责展示和交互，后端负责数据处理和业务逻辑。通过Spring Boot框架，实现了高效、可扩展的会员信息管理系统。
